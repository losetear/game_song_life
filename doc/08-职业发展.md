## 一、务农

> "春种一粒粟，秋收万颗子"

### 入门条件
- 无门槛，任何人都可以种地
- 需要有田地（自有/租佃）
- 需要农具（锄头、犁等）和种子

### 成长阶段

| 阶段 | 称号 | 条件 | 解锁内容 |
|------|------|------|---------|
| 1 | 佃农 | 租田 ≤ 5亩 | 基础种植，交租后所剩无几 |
| 2 | 自耕农 | 自有田 5-20亩 | 选种优质品种，余粮可卖 |
| 3 | 富农 | 自有田 20-50亩 | 可雇短工，种植经济作物 |
| 4 | 地主 | 自有田 ≥ 50亩 | 雇长工管理，收租为主，可兼营他业 |

### 日常活动
```
春：翻地 → 育秧 → 插秧 → 施肥
夏：除草 → 引水 → 除虫 → 防涝
秋：收割 → 晾晒 → 入仓 → 交税/卖粮
冬：休整 → 修补农具 → 赶集 → 计划来年
```

### 收入来源
- 粮食售卖（价格随季节波动：秋收最低，青黄不接时最高）
- 经济作物（棉花、麻、茶叶）利润更高但需技能
- 出租田地收租（成为地主后）

### 特殊事件
- **丰年**：产量大增，但粮价下跌，可能"谷贱伤农"
- **灾年**：旱涝虫灾，颗粒无收，需要救济或借粮
- **蝗灾**：突发性毁灭，需全村合力扑杀
- **官府征粮**：额外加税，可能被迫卖地
- **田产纠纷**：邻居争地界，需打官司或私了

### 风险与代价
- 天灾不可控，一次就可能倾家荡产
- 体力劳动繁重，体质消耗大
- 被地主压榨的佃农很难翻身
- 税赋沉重，丰年也未必富裕

### 转职路线
- 攒够钱 → 经商（卖粮起家）
- 有余粮 → 开酒坊/豆腐坊（手艺人）
- 交到官府人脉 → 当里正/保长（衙役系统）
- 种植草药 → 转行医术

## 二、捕鱼

> "孤舟蓑笠翁，独钓寒江雪"

### 入门条件
- 需要鱼竿/渔网/小舟（看规模）
- 需要找到水域（码头区、汴河、溪流）

### 成长阶段

| 阶段 | 称号 | 条件 | 解锁内容 |
|------|------|------|---------|
| 1 | 垂钓新手 | 有鱼竿 | 岸边垂钓，产量低 |
| 2 | 熟手渔夫 | 钓鱼技能 ≥ 熟练 | 可撒网捕鱼，产量提升 |
| 3 | 船家 | 拥有小舟 | 可到河中央下网，产量大增 |
| 4 | 鱼行把头 | 拥有渔船+雇工 | 批量捕鱼，控制鱼市 |

### 日常活动
- 清晨出发，选择捕鱼地点
- 不同地点鱼种不同（码头=杂鱼、上游=鲤鱼、溪流=鳟鱼）
- 上午捕鱼，午间去鱼市售卖
- 修补渔网、保养渔船

### 收入来源
- 鱼市卖鱼（直接卖给鱼贩或摆摊）
- 供应酒楼（稳定但价格被压）
- 珍稀鱼种（偶尔捕到，可高价出售）
- 虾蟹贝类（季节性）

### 特殊事件
- **鱼汛**：某季节鱼群大量出现，忙碌但丰收
- **水匪**：汴河下游有水匪出没，被劫则损失惨重
- **风暴**：河上突发大风浪，小船可能翻覆
- **禁渔令**：官府偶尔颁布禁渔（如祭祀期间）
- **溺水**：不会游泳时落水的致命风险

## 三、狩猎

> "林暗草惊风，将军夜引弓"

### 入门条件
- 需要武器（弓箭、陷阱、猎叉）
- 需要进入山林区域
- 建议有向导或拜师学艺

### 成长阶段

| 阶段 | 称号 | 条件 | 解锁内容 |
|------|------|------|---------|
| 1 | 初猎者 | 有基础武器 | 可猎兔、鸟等小型猎物 |
| 2 | 猎手 | 狩猎技能 ≥ 熟练 | 可猎鹿、野猪，布设陷阱 |
| 3 | 资深猎人 | 狩猎技能 ≥ 精通 | 可追踪大型猎物，识别兽径 |
| 4 | 猎王 | 狩猎技能 ≥ 大师 | 可猎虎豹，受官府征召除害 |

### 日常活动
- 检查/布设陷阱
- 追踪猎物痕迹
- 等待伏击或主动搜寻
- 处理猎物（剥皮、分割、腌制）
- 到城中售卖皮毛和兽肉

### 收入来源
- 兽肉（卖给肉铺/酒楼）
- 皮毛（卖给皮货商，价格看品质）
- 药材（猎物身上偶尔获得的珍贵材料）
- 官府悬赏（猛兽伤人时官府出赏金）

### 特殊事件
- **猛兽袭人**：虎狼出没伤及村民，官府悬赏
- **山贼遭遇**：山林深处遇到匪帮，需周旋
- **发现珍稀动物**：白鹿、金雕等，皮毛价值极高
- **迷路**：大雾/深山迷路，消耗体力寻找出路
- **猎物反杀**：野猪/虎负隅顽抗，可能受伤甚至死亡

### 风险
- 山林危险度高，野生动物可能致命
- 偏远地区无救援
- 冬季山林更加危险
- 可能与山贼发生冲突

## 四、科举仕途

> "十年寒窗无人问，一举成名天下知"

### 入门条件
- 才学属性 ≥ 20
- 需要购买书籍/拜师
- 需要大量时间投入

### 成长阶段

| 阶段 | 功名 | 考试 | 声望 | 解锁 |
|------|------|------|------|------|
| 1 | 童生 | 无（基础） | +5 | 可在私塾教书糊口 |
| 2 | 秀才 | 县试（才学≥40） | +15 | 免徭役，见官不跪，可教书 |
| 3 | 举人 | 乡试（才学≥60，每三年一次） | +40 | 可做官（候补），社会地位大增 |
| 4 | 进士 | 会试+殿试（才学≥80，每三年一次） | +80 | 授官职，进入仕途 |
| 5 | 朝臣 | 政绩积累+政治斗争 | +100 | 可做知府乃至京官 |

### 日常活动

**备考阶段**
```
辰时 → 起床读书
巳时 → 继续研读经义/练字
午时 → 午饭，短暂休息
未时 → 写文章练习
申时 → 与同窗论学/拜访名师
酉时 → 复习当日所学
戌时 → 灯下夜读
亥时 → 就寝
```

**考试过程**
- 考前需准备笔墨纸砚
- 进考场（贡院），封闭数日
- 考试内容由才学值+随机因素决定成败
- 考后等待放榜，期间焦虑不安

### 收入来源
- **童生/秀才**：教书（私塾先生）、代写书信、卖字
- **举人**：被聘为幕僚、受富户请托办事
- **进士/官员**：朝廷俸禄，灰色收入看个人操守

### 特殊事件
- **考场舞弊**：被发现则革除功名，永不录用
- **恩科**：皇帝特批加考一次，机会难得
- **落榜**：打击巨大，心情暴跌，可能一蹶不振
- **名师提携**：受到某位大儒赏识，才学暴涨
- **朋党之争**：中举后卷入政治派系斗争
- **外放做官**：被派往偏远地区任职
- **被弹劾**：做官期间可能被政敌攻击

### 风险
- 投入巨大但成功率极低（科举录取率约1-3%）
- 长期读书无法赚钱，经济压力大
- 年龄增长后记忆力和精力下降
- 一旦卷入党争可能丢官甚至丧命
- 落榜后的心理打击可能导致极端行为

### 转职路线
- 屡考不中 → 师爷/幕僚（为官员出谋划策）
- 屡考不中 → 行医（转学术为医术）
- 屡考不中 → 落草（极端情况下对体制失望）
- 中举后做官 → 贪官路线（收受贿赂，高风险高回报）
- 中举后做官 → 清官路线（声望高但穷）

### 分支职业

| 分支 | 技能 | 产品 | 市场需求 |
|------|------|------|---------|
| 木匠 | 木工 | 家具、建筑构件、农具 | 稳定 |
| 铁匠 | 锻造 | 刀具、农具、铁器 | 稳定 |
| 织工 | 织布 | 麻布、棉布 | 大 |
| 裁缝 | 缝纫 | 衣服 | 稳定 |
| 陶工 | 制陶 | 陶器、瓷器 | 中等 |
| 造纸匠 | 造纸 | 纸张 | 中等 |
| 厨子 | 烹饪 | 食物/酒菜 | 大 |
| 酿酒师 | 酿造 | 酒 | 大 |
| 漆匠 | 漆艺 | 漆器 | 小众高价 |
| 金银匠 | 金银加工 | 首饰 | 小众高价 |

### 成长阶段

| 阶段 | 称号 | 条件 | 解锁 |
|------|------|------|------|
| 1 | 学徒 | 拜师，技能入门 | 做粗活，学基础，无收入 |
| 2 | 出师 | 技能 ≥ 熟练，师傅认可 | 可独立干活，接简单订单 |
| 3 | 师傅 | 技能 ≥ 精通 | 可开铺收徒，接复杂订单 |
| 4 | 名匠 | 技能 ≥ 大师 | 作品可高价出售，受官府/富户定制 |
| 5 | 宗师 | 技能满级 + 特殊成就 | 作品传世，声望极高 |

### 学徒期设计
```
学徒期的日常：
  辰时 → 到师傅铺子报到
  巳时 → 做杂活（搬运、清理）
  午时 → 师傅吃饭，学徒吃剩的
  未时 → 观摩师傅操作
  申时 → 在师傅指导下练习基础
  酉时 → 收拾铺子，回家

  学徒期一般持续 1-3年（游戏内）
  期间无收入，但技能逐步提升
  师傅好感度高 → 教得更快
  师傅好感度低 → 可能被刁难/延长学徒期
```

### 收入来源
- 接订单（按件计费）
- 摆摊售卖产品
- 开铺面坐等客人
- 供应大宗订单（酒楼、官府）
- 名匠可接定制高价单

### 厨子/酿酒师特殊路线

**厨子**
- 可在酒楼当厨师（稳定薪水）
- 可自己摆摊卖小吃（灵活但辛苦）
- 可开饭馆（需要本金，收益最高）
- 名厨可被官府/富户聘请（高收入）
- 宋朝名菜谱：东坡肉、宋嫂鱼羹、蟹黄汤包等

**酿酒师**
- 需要粮食/水果作为原料
- 酿造需要时间（数日到数月）
- 可供应酒楼或自己开店
- 好酒可卖出高价
- 受官府酒税管控

## 六、经商

> "天下熙熙，皆为利来；天下攘攘，皆为利往"

### 分支路线

| 路线 | 特点 | 资本要求 | 风险 |
|------|------|---------|------|
| 行商（贩夫走卒） | 走街串巷，低买高卖 | ★☆☆ | ★★☆ |
| 摊贩 | 夜市/早市固定摆摊 | ★★☆ | ★★☆ |
| 坐商（开店） | 租/买铺面，稳定经营 | ★★★★ | ★★★ |
| 批发商 | 大宗货物，走量 | ★★★★★ | ★★★★ |
| 合伙经商 | 与人合股，共担风险 | ★★★ | ★★★ |
| 海商/远洋 | 远距离贸易，利润极高 | ★★★★★ | ★★★★★ |

### 成长阶段

| 阶段 | 称号 | 条件 | 月收入参考 |
|------|------|------|-----------|
| 1 | 货郎 | 本金 ≤ 500文 | 100-300文 |
| 2 | 小贩 | 本金 500-5000文，有固定摊位 | 300-1000文 |
| 3 | 商户 | 拥有店铺，本金 5000-50000文 | 1-3贯 |
| 4 | 富商 | 多家产业，本金 ≥ 50000文 | 5-20贯 |
| 5 | 巨贾 | 行业龙头，资产 ≥ 500贯 | 20贯以上 |

### 经商核心玩法 — 供需系统

```
商品供需循环：
  产地（农田/山林/作坊）→ 批发商 → 零售商 → 消费者

影响价格的因素：
  ┌── 季节：冬天的蔬菜比夏天贵
  ├── 天气：暴雨后蔬菜涨价
  ├── 节日：过年/中秋需求激增
  ├── 灾害：灾年粮价飞涨
  ├── 政策：官府调控/禁令
  └── 竞争：同类型商铺多则价格下降

玩家可以做的：
  - 判断行情，囤货待涨
  - 找低价货源（产地直采）
  - 开发新市场（没人卖的东西）
  - 垄断某种商品（控制供给）
```

### 商业活动详解

**摆摊**
```
前置：有可卖的商品 + 摊位费（夜市10-50文/晚）

流程：
  1. 选择摆摊地点（人流量不同）
  2. 选择售卖商品
  3. 设定价格（看讨价还价技能）
  4. 等待顾客（消耗时间）
  5. 打烊算账

影响收入因素：商品稀缺度、定价、地段、天气、时辰
```

**开店**
```
前置：本金 ≥ 3000文（租金+装修+进货）

流程：
  1. 选址租铺（位置决定客流和租金）
  2. 装修铺面
  3. 进货上架
  4. 雇伙计（可选，工资约300-500文/月）
  5. 日常经营

开销：租金 + 伙计工资 + 进货成本 + 商税
收入：商品售价 × 销量
利润 = 收入 - 开销（需要精打细算）
```

**行商（贩运）**
```
前置：本金 + 货物 + 运输工具（推车/牛车/马匹）

流程：
  1. 在产地低价进货
  2. 运输到缺货的地方
  3. 高价卖出

风险：
  - 路上遇山贼（可能被劫）
  - 天气恶劣（货物损坏）
  - 到了目的地发现价格不如预期
  - 运输成本吃掉利润

利润率：50%-300%（看眼光和运气）
```

### 行会系统

宋朝各行有行会，不入行则难以正经做生意。

```
行会规则：
  - 加入需推荐人 + 入会费
  - 行会规定价格区间（不许恶意降价）
  - 行会保护成员利益（对付地痞/官府）
  - 行会头领（行首）有很大权力

  不入行的后果：
  - 被排挤，其他商人不跟你做生意
  - 没有行会保护，容易被欺负
  - 官府不认可，难以正规经营
```

### 特殊事件
- **官府采买**：大订单机会，但需人脉和实力
- **商战**：与同行竞争，可正当竞争也可使手段
- **走私**：违禁品（盐、茶、酒）利润极高但风险极大
- **合伙被骗**：合伙人卷款跑路
- **商铺失火**：意外灾害，损失惨重
- **官府勒索**：被贪官盯上，需打点

## 七、山贼/绿林

> "此山是我开，此树是我栽，要想从此过，留下买路财"

### 入门条件
- 声望 ≤ -20（被社会排斥者更容易入伙）
- 或走投无路（负债/犯罪被通缉/饥荒无食）
- 需要找到山寨投靠或自己建立山寨

### 成长阶段

| 阶段 | 称号 | 条件 | 势力 |
|------|------|------|------|
| 1 | 嘛喽 | 刚入伙 | 无权力，听命行事 |
| 2 | 头目 | 武力+声望（黑道）积累 | 手下3-5人，可单独行动 |
| 3 | 寨主 | 威望够高，前任让位或挑战上位 | 控制整个山寨 |
| 4 | 绿林豪杰 | 势力大，名声响 | 多个山寨联盟 |
| 5 | 割据一方 | 拥有武装力量 | 官府忌惮，可能被招安 |

### 日常活动

**嘛喽阶段**
```
  白天 → 在山寨里放哨/打杂
  傍晚 → 跟头目下山劫道
  夜间 → 分赃、喝酒、赌钱
  不定时 → 去附近村落"收保护费"
```

**寨主阶段**
```
  管理 → 分配任务、处理内部纠纷
  经营 → 销赃渠道、招兵买马
  战略 → 选择劫掠目标、避开官府锋芒
  外交 → 与其他山寨结盟/对抗、与贪官勾结
```

### 收入来源
- **劫道**：在山路上设卡收费或直接抢劫
- **收保护费**：向附近村落收取"平安钱"
- **绑票**：绑架富户家人勒索赎金
- **销赃**：将劫来的货物通过黑市变现
- **走私**：控制山口要道，收取过路费

### 山寨经营
```
山寨是一个微型据点，需要管理：
  ├── 粮食储备（需要定期下山抢粮或买粮）
  ├── 兵器装备（需要铁匠或抢夺）
  ├── 人员士气（分赃公平、有酒有肉）
  ├── 防御工事（工事等级影响防御力）
  ├── 情报网络（山下的眼线）
  └── 销赃渠道（城里的黑市商人）
```

### 特殊事件
- **官府围剿**：衙门派兵攻打山寨，需要防御或转移
- **黑吃黑**：其他山贼来抢地盘
- **招安**：官府许以官职，招安归顺（转职衙役）
- **内讧**：手下不服，可能叛变
- **百姓围攻**：如果太过分，村民可能联合起来反抗
- **英雄好汉**：有侠义之士上山挑战

### 风险
- 永久通缉，进不了城（除非乔装）
- 随时可能被官府围剿
- 内部可能叛变
- 生病受伤无医可看
- 一旦被抓，重则死刑

### 招安路线
```
山贼 → 积累势力 → 官府注意 → 谈判招安
  → 交出部分人马 → 获得低级武职
  → 声望重置（从负变为低正）
  → 可以入城但受到歧视

  或拒绝招安 → 继续占山 → 最终被围剿
```

## 八、水匪/海盗

> "浪里白条，水上豪杰"

### 入门条件
- 需要水性（不会游泳很危险）
- 需要船只
- 通常在码头区或汴河下游活动

### 成长阶段

| 阶段 | 称号 | 条件 | 活动范围 |
|------|------|------|---------|
| 1 | 水贼 | 入伙或单干 | 汴河近郊，劫小船 |
| 2 | 船主 | 有自己的船+几个手下 | 汴河中下游 |
| 3 | 水寨头领 | 多条船，有据点 | 河道/湖泊 |
| 4 | 水上霸主 | 船队规模大 | 江河湖海 |

### 日常活动
- 在河湾/险滩设伏
- 劫掠过往商船
- 收取水路保护费
- 在岸边销赃补给

### 收入来源
- 劫掠商船货物
- 收取"平安通行费"
- 走私（水上运输违禁品）
- 鱼获（伪装成渔民时）

### 与山贼的区别
```
水匪优势：
  ✓ 水上机动性强，官府难以追捕
  ✓ 可伪装成渔民
  ✓ 控制水路交通要道
  ✓ 船是移动据点

水匪劣势：
  ✗ 受天气影响大（暴风雨致命）
  ✗ 旱季水位低时活动受限
  ✗ 船只维护成本高
  ✗ 水上战斗有溺水风险
```

### 特殊事件
- **暴风雨**：船翻人亡，最致命的风险
- **官府水师**：朝廷调水军围剿
- **遇上硬茬**：劫了镖局的船，引来报复
- **发现沉船**：水下打捞，可能获得意外之财
- **迷雾**：水上大雾，可能迷失方向或撞上暗礁

### 灰色收入（贪腐路线）

### 特殊事件

## 十、行医

> "悬壶济世，医者仁心"

### 入门条件
- 需拜师学医（医术技能 ≥ 入门）
- 或自学医书（才学 ≥ 30 + 购买医书）

### 成长阶段

| 阶段 | 称号 | 医术等级 | 可治疾病 |
|------|------|---------|---------|
| 1 | 药童 | 入门 | 认药、抓药、简单外伤 |
| 2 | 郎中 | 熟练 | 风寒、腹泻、跌打损伤 |
| 3 | 名医 | 精通 | 疑难杂症、接骨、解毒 |
| 4 | 国手 | 大师 | 基本什么都能治，名声远扬 |

### 日常活动
```
  辰时 → 药铺坐堂，等待病人
  巳-午 → 看诊、开方、抓药
  未时 → 出诊（到病人家中）
  申时 → 采药/整理药柜
  酉时 → 研读医书
  不定时 → 紧急救治
```

### 收入来源
- 诊金（10-100文/次，看名气）
- 卖药利润
- 富户请诊（出诊费高）
- 官府聘用（做官医）

### 特殊事件
- **瘟疫**：大范围传染病，忙不过来
- **治死人**：误诊导致病人死亡，可能被追责
- **官府征召**：被征为军医或太医院候选
- **发现新药方**：偶然发现特效药，名声大增
- **遇到疑难杂症**：治好则名声大增，治不好则声誉受损

### 药材系统
```
药材来源：
  ├── 自采：上山采药（需要辨识技能）
  ├── 收购：从药农/猎人手中收购
  ├── 进货：从药市批发
  └── 炮制：将生药加工为成药

常见药材：
  甘草、黄芪、当归、白芍、川芎、茯苓、
  半夏、陈皮、麻黄、桂枝、石膏、芒硝...

  药方组合：
  不同药材组合产生不同疗效
  配错可能加重病情甚至害命
```

## 十五、兼职与转职系统

### 兼职规则

玩家不限于一个职业，可以同时从事多个：

| 兼职组合 | 可行性 | 说明 |
|---------|--------|------|
| 农夫 + 猎户 | ✓ | 农闲时上山打猎 |
| 农夫 + 渔民 | ✓ | 田边有河就顺带捕鱼 |
| 书生 + 代写书信 | ✓ | 没考中时赚生活费 |
| 商贩 + 手艺人 | ✓ | 自己做自己卖 |
| 商贩 + 走私 | ✓（风险） | 正常生意掩护走私 |
| 镖师 + 商贩 | ✓ | 走镖同时带货 |
| 僧人 + 行医 | ✓ | 僧医自古有之 |
| 衙役 + 收保护费 | ✓（贪腐） | 灰色地带 |
| 山贼 + 水匪 | ✗ | 不在同一地理区域 |
| 书生 + 山贼 | 特殊 | "逼上梁山"剧情线 |

### 转职触发条件

```
转职不是点按钮，而是由事件/条件自然触发：

主动转职：
  "我想开店" → 需要本金 + 找到铺面 + 入行会
  "我想考科举" → 需要才学 + 购买书籍
  "我想出家" → 去寺庙申请（方丈考核）
  "我想拜师学艺" → 找到愿意收徒的NPC

被动转职（被环境所迫）：
  "田产被没收" → 农夫失去土地，需找新出路
  "负债被追" → 可能逃入山林成为山贼
  "犯事被通缉" → 只能亡命天涯
  "科举屡败" → 可能放弃转行
  "被招安" → 山贼变为官军

转职代价：
  - 原职业积累的资源可能部分丢失
  - 新职业需要从低级开始
  - 声望可能受影响（商人转山贼声望暴跌）
```

### 人生轨迹示例

```
[路线A：寒门贵子]
  农家子弟 → 省吃俭用买书 → 苦读十年 → 中秀才 → 中举人 → 进士及第 → 外放知县 → 政绩卓著 → 升任知府

[路线B：商海沉浮]
  商贩之后 → 跟父亲摆摊 → 继承小铺 → 扩大规模 → 开分号 → 被贪官盯上 → 行贿自保 → 越陷越深 → 事发被抄家

[路线C：逼上梁山]
  落魄书生 → 屡考不中 → 被债主追债 → 逃入山林 → 入伙山贼 → 凭才学成为军师 → 势力壮大 → 被招安 → 获得低级武职

[路线D：医者仁心]
```
