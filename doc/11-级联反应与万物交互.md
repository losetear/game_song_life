# 一、级联反应系统

## 1.1 核心理念

所有变化通过**事件**传播，一个事件可触发多个系统的连锁响应。这是"事事有回应"的技术实现核心。

## 1.2 事件类型

| 事件类型 | 说明 | 触发源 |
|----------|------|--------|
| weather_changed | 天气变化 | 天气系统 |
| time_advanced | 时间推进 | 时间系统 |
| season_changed | 季节变化 | 时间系统 |
| entity_created | 实体创建 | 世界管理器 |
| entity_destroyed | 实体销毁 | 多个系统 |
| player_action | 玩家操作 | 交互系统 |
| npc_behavior | NPC行为 | NPC AI |
| npc_interaction | NPC间互动 | NPC AI |
| economy_transaction | 交易 | 经济系统 |
| area_entered | 进入区域 | 移动系统 |
| festival_started | 节日开始 | 时间系统 |
| disaster_triggered | 灾害触发 | 天气系统 |

## 1.3 级联控制参数

| 参数 | 值 | 说明 |
|------|------|------|
| 最大传播深度 | 10层 | 超过自动截断 |
| 每层效果衰减 | ×0.7 | 越远影响越小 |
| 每 Tick 事件预算 | 500个 | 超出留到下个 Tick |
| 去重 | 同事件+同实体同Tick | 只触发一次 |
| 单系统限流 | 100个子事件/Tick | 超过限流该系统 |

## 1.4 级联示例——一场暴雨

```
天气系统发出 "weather_changed" { type: "heavy_rain" }

Tier 0 — 即时反应（< 2秒）
├── 地面：所有户外地块湿度+50，石板路变"湿滑"
├── NPC：户外 NPC 评估避雨
│   ├── 有伞 → 撑伞
│   ├── 无伞 → 跑向最近屋檐
│   └── 商人 → 收摊
├── 动物：鸟归巢、猫躲雨、鱼活跃
├── 物品：露天物品受潮（布匹质量下降、火堆熄灭）
└── 叙述：生成雨景文字描述

Tier 1 — 短期影响（10秒-1分钟游戏时间）
├── 经济：伞需求×3（涨价），户外商品供给×0.5
├── 农业：所有农田水分+30，作物生长加速
├── 地形：低洼处积水，泥土路变泥泞
├── NPC：淋雨的NPC可能感冒 → 去药铺买药
└── 记忆：NPC 记住"那场暴雨"

Tier 2 — 长期影响（小时/天）
├── 持续暴雨 → 洪水风险
│   ├── 低洼地区被淹
│   ├── 农作物受损
│   └── 官府组织抗洪
├── 河流水位上涨 → 渡船可能停运 → 影响贸易路线
└── 生态：雨水充足 → 植物茂盛 → 来年丰收可能
```

## 1.5 级联示例——砍掉一棵古树

```
玩家砍伐古银杏

即时：
├── 获得：银杏木×20（上等材料）
├── 惊飞树上鸟群
└── 巨响引起注意

NPC 反应：
├── 村里老人：愤怒，好感-40
├── 以前在树下休息的人：不满，好感-10
├── 口碑传播："有人砍了古树"（传播范围大）
└── 可能被罚款 50-200文

环境：
├── 该区域鸟类-50%（失去栖息地）
├── 如靠近河岸，水土流失风险+30
└── 夏天这条路没有树荫了

不可逆：这棵树永远没了
```

# 二、万物交互设计

## 2.1 交互层级

| 层级 | 操作类型 | 是否改变对象状态 | 示例 |
|------|----------|----------------|------|
| Level 0 | 被动感知 | 否 | 对象存在于世界中 |
| Level 1 | 观察（看/听/闻/摸） | 否 | "这是一棵百年银杏，叶已转黄" |
| Level 2 | 操作（拿/放/用/组合/破坏） | 是 | 折下柳枝→柳枝进背包+柳树状态变化 |
| Level 3 | 关系（送/交易/偷/修） | 是+影响NPC | 送帽子给邻居小孩→好感+5 |
| Level 4 | 叙事（触发世界变化） | 不可逆 | 砍古树→村民集体愤怒→可能被罚款 |

## 2.2 上下文感知

交互按钮根据以下维度动态变化：

| 维度 | 示例 |
|------|------|
| 手持物品 | 拿着鱼→猫出现"喂鱼"；拿着斧头→树出现"砍伐" |
| 角色技能 | 学了点茶→茶具出现"点茶（表演）" |
| NPC 状态 | NPC在忙→"闲聊"变灰 |
| 天气 | 下雨→"避雨" |
| 时间 | 夜晚→建筑出现"敲门" |
| 好感度 | 好友→"借钱""学手艺" |
| 季节 | 秋天→银杏"拾落叶" |

## 2.3 植物交互列表

| 交互 | 条件 | 效果 |
|------|------|------|
| 细看 | 无 | 描述树种/状态/季节 |
| 摸树皮 | 无 | 触感描述 |
| 摘叶 | 有叶子 | 获得1片叶子 |
| 拾落叶 | 地面有落叶 | 获得1-5片落叶 |
| 折枝 | 体魄>15 | 获得树枝，树健康-2 |
| 攀爬 | 体魄>30 | 到达高处观察（可能摔落） |
| 系绳 | 手持绳子 | 绳子绑树上（可拴船/马） |
| 题字 | 手持刀具 | 刻字（其他NPC看到并评论） |
| 砍伐 | 手持斧头+砍伐技能Lv10 | 获得大量木材+树消失+级联反应 |
| 在树下休息 | 树冠够大 | 恢复体力 |
| 避雨 | 下雨+树冠大 | 暂时躲避小雨 |
| 采果 | 结果季节 | 获得果实 |
| 浇水 | 手持水具 | 水分+20 |
| 施肥 | 手持肥料 | 养分+20 |

## 2.4 动物交互列表

### 猫/狗

| 交互 | 条件 | 效果 |
|------|------|------|
| 观察 | 无 | 描述品种/状态/行为 |
| 招手 | 距离近 | 信任度判定：靠近或跑开 |
| 投喂 | 手持食物 | 信任+5~+15 |
| 抚摸 | 信任>30 | 信任+3或被抓（信任低） |
| 抱起 | 信任>60 | 可携带 |
| 取名 | 信任>50 | 显示自定义名字 |
| 收养 | 信任>70 | 成为宠物（跟随+住处增加猫窝） |
| 追赶 | 无 | 信任-10 |

### 野生动物

| 交互 | 条件 | 效果 |
|------|------|------|
| 观察 | 无 | 描述（感知判定，远距离可能看不到） |

# 三、叙述系统

## 3.1 设计规则

每个交互/事件对应一个叙述模板，包含多条文案，运行时加权随机选择。

**感知属性影响叙述丰富度**：

| 感知范围 | 示例 |
|----------|------|
| 0-30 | "一棵{树种}，{季节状态}。" |
| 30-60 | "这是一棵{树种}，{高度}，{健康状态}。{季节描述}。" |
| 60-100 | "这棵{树种}大约{年龄估计}，树干{细节}。{季节详细描述}" |

## 3.2 叙述示例

### 砍树（3条备选）

| 权重 | 文案 |
|------|------|
| 3 | "你抡起斧头，对准{树名}的根部用力劈下。木屑飞溅，沉闷的声响在周围回荡。约莫一炷香后，大树终于支撑不住，轰然倒下。鸟群惊叫着飞散。" |
| 2 | "斧刃一次次嵌入{树名}的躯干，每一下都带出新鲜的木屑。终于，这棵大树发出一声叹息般的断裂声，缓缓倒向地面。" |
| 1 | "{树名}在斧头下颤抖，像是最后一次反抗。但终究敌不过利刃，倒下的瞬间惊起了一群栖息的{鸟名}。" |

### 下雨开始

| 权重 | 文案 |
|------|------|
| 3 | "天色渐暗，几滴雨点打在你脸上。抬头望去，乌云已不知不觉吞没了半边天。街上的行人开始加快脚步。" |
| 2 | "空气中弥漫着泥土的气息，然后雨就来了。先是零星几点，转眼间便成瓢泼之势。" |

# 四、环境与生态系统

## 4.1 天气影响总览

| 天气 | NPC行为 | 物品 | 经济 | 生态 |
|------|---------|------|------|------|
| 晴天 | 正常 | 正常 | 正常 | 植物蒸发快 |
| 下雨 | 避雨/撑伞 | 露天物品受潮 | 伞涨/户外货跌 | 植物获得水分 |
| 暴雨 | 大部分回家 | 严重受潮 | 运输中断 | 可能洪涝 |
| 下雪 | 减少外出 | 水结冰 | 燃料涨价 | 植物休眠 |
| 大雾 | 行动谨慎 | 正常 | 渡船停运 | — |

## 4.2 生态食物链

```
高级捕食者：虎 → 鹿/野猪
中型捕食者：狼 → 兔/鸡 | 狐狸 → 兔/鸡
小型动物：兔 → 草 | 老鼠 → 谷物
昆虫类：蜜蜂 → 花蜜 | 蛐蛐 → 植物
水生类：大鱼 → 小鱼 → 虾/虫
```

**生态平衡规则**：
- 砍掉大树 → 鸟类减少 → 虫害增加 → 农作物受损
- 猎杀过多狼 → 兔子暴增 → 菜地被啃
- 过度捕鱼 → 鱼群减少 → 渔夫收入下降
- 保护生态是一种长期策略

## 4.3 动物种群动态（L2统计规则）

```
每日更新：
  当前数量 = 上日数量 × (1 + 繁殖率 - 狩猎压力 - 自然死亡)

  繁殖率：兔0.05, 鹿0.03, 野猪0.02, 狼0.01, 虎0.005
  自然死亡：0.01（所有种类）
  狩猎压力：取决于区域内猎人数 × 狩猎技能

  数量上限 = 区域容量上限
  数量 < 最低阈值 → "此区域动物稀少"
```

## 4.4 植物生长公式

```
每Tick生长量 = 基础生长率 × 品种因子 × 水分因子 × 养分因子 × 温度因子 × 健康因子

水分因子：
  moisture < 20: ×0.2（严重缺水）
  moisture 20-40: ×0.5
  moisture 40-80: ×1.0（适宜）
  moisture > 80: ×0.8（过多）
```
