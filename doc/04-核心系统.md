# 04 — 核心系统

## 一、经济系统

### 货币
- **文钱**：基础货币，日常交易
- **贯/吊**：1000文 = 1贯，大宗交易
- **银两**：1两 ≈ 1-2贯（浮动），高价值交易

### 物价参考（宋朝风格）

| 物品 | 价格 | 备注 |
|------|------|------|
| 一个炊饼 | 5-10文 | 便宜填肚子 |
| 一碗羊肉汤 | 30-50文 | 体面一餐 |
| 酒楼一顿 | 100-300文 | 看档次 |
| 一匹麻布 | 200-500文 | 做衣服用 |
| 一石米 | 300-800文 | 看年成波动 |
| 一头耕牛 | 5000-10000文 | 大资产 |
| 城中小院 | 500-1000贯 | 需多年积累 |

### 商业活动

**打工**
- 码头搬运、店铺伙计、账房先生等
- 日结工钱，稳定但收入低

**摆摊**
- 夜市/早市可摆摊
- 需要有可卖的东西（食物、手工艺品等）
- 收入看手艺和客流

**开店**
- 需要本金：租金、进货、装修
- 可雇伙计帮忙
- 收入可观，但有风险

**倒买倒卖**
- 利用信息差/季节差价
- 低买高卖
- 需要本钱和人脉

### 经济平衡机制（原稿缺失，补充）

**防止通胀的设计**

| 机制 | 说明 |
|------|------|
| 税收 | 商业活动需交税，大额交易有契税 |
| 物品损耗 | 工具/衣物会磨损，需维修或更换 |
| 食物消耗 | 必须持续消耗食物，是主要开销 |
| 住房成本 | 租房/买房/修缮都需要钱 |
| 医疗支出 | 生病治疗费用可观 |
| 社交支出 | 维护关系需要请客送礼 |
| 意外损失 | 被偷、被骗、天灾损失 |

**钱 Sink 设计**

```
日常开销（持续）
├── 食物（必需）
├── 住房（必需）
└── 工具维护（半必需）

阶段性开销（偶发）
├── 医疗
├── 人情往来
├── 购买升级装备
└── 纳税

大额开销（稀有）
├── 买房/买田
├── 科举赶考
├── 开店
└── 婚嫁
```

**物价波动逻辑**
- 基础物价 + 季节修正 + 供需修正 + 随机波动
- 丰收年粮价下跌，灾年上涨
- 商人囤货会导致短期短缺和价格上涨

> **设计目标**：玩家中后期也不会"钱没处花"，总有追求的财务目标。

## 二、官府与律法系统

### 纳税
- **夏税秋粮**：有田产者每年两次纳税
- **商税**：经商需交税，过卡要交关税
- **徭役**：可能被征调做工（可出钱雇人替）

### 科举
```
童试（县试）→ 秀才
乡试（省试）→ 举人
会试 + 殿试 → 进士
```
- 考试需要才学达标
- 中举后声望大涨，有机会当官
- 落榜可再考，但有年龄压力

### 犯罪与惩罚

| 行为 | 后果 |
|------|------|
| 偷窃 | 轻则罚款/打板子，重则刺配流放 |
| 斗殴 | 罚款/监禁，伤人加重 |
| 杀人 | 死刑/秋后问斩 |
| 逃税 | 罚款+补缴，声望下降 |
| 贿赂 | 被发现则双方受罚 |

> 夜间入室、走暗巷可能被偷。被偷可报官，但能否追回看衙役能力。

### 公职
- **衙役**：巡逻、抓贼，月有俸禄
- **书吏**：衙门文书工作
- **教头**：教武艺，需有武艺技能
- 考中举人/进士后可当官（高级玩法）

## 三、店铺系统

### 店铺类型

| 店铺 | 功能 | 特殊 |
|------|------|------|
| 粮铺 | 买米面粮油 | 价格随季节波动 |
| 布庄 | 买布料 | 可定制衣服 |
| 药铺 | 买药/看病 | 坐堂大夫可看病 |
| 铁匠铺 | 买刀具农具 | 可定制 |
| 书店 | 买书 | 提升才学 |
| 酒楼 | 吃饭社交 | 最贵的餐饮 |
| 茶馆 | 喝茶听消息 | 消息灵通之地 |
| 当铺 | 典当/赎回 | 急用钱可当东西 |
| 钱庄 | 存钱/借钱 | 有利息 |
| 客栈 | 住宿 | 无房时可住 |

### 店铺交互
- 每个店铺有店主NPC，性格各异
- 可讨价还价（看技能）
- 可与店主建立关系，获优惠或消息
- 店铺有营业时间，关门后无法交易

## 四、犯罪系统（玩家视角）

### 偷窃（可行但不推荐）
- 需在无人/暗处进行
- 有被发现的概率（看技能、环境、NPC警觉）
- 成功可获得财物
- 失败则声望大损，可能被扭送官府

### 行骗
- 可以次充好、设局骗人
- 成功获利，失败声望扫地
- 被骗NPC可能记仇

### 反犯罪（玩家作为受害者）
- 家中被偷 → 报官/自己调查/雇人追查
- 被骗 → 报官/找人帮忙/认栽
- 夜间遇劫 → 选项：跑/打/交钱/呼救

## 五、自然生态

> 万级规模下，自然生态分为**个体级**（玩家视野内）和**区域级**（远处统计模拟）。
> 详见 [08-技术架构.md](08-技术架构.md) 的LOD章节。

### 动物

**家畜（城郊/农村）— 个体实体，L1模拟**
| 动物 | 可交互 | 产出 | 区域容量参考 |
|------|--------|------|------------|
| 鸡 | 喂养、宰杀 | 蛋、肉 | 每户3-10只 |
| 鸭 | 喂养、宰杀 | 蛋、肉 | 每户2-5只 |
| 猪 | 喂养、宰杀 | 肉 | 每户1-3头 |
| 牛 | 喂养、租用、宰杀（犯法） | 耕田、牛奶 | 每10户1头 |
| 羊 | 喂养、宰杀 | 羊毛、羊肉 | 每户2-5只 |
| 马 | 喂养、骑乘 | 骑乘、拉车 | 富户/官府专用 |
| 狗 | 喂养、驯养 | 看家护院 | 每户0-2只 |
| 猫 | 喂养 | 抓老鼠、陪伴 | 每户0-1只 |

**野生动物（山林/河流）— 远处L2统计，近处L0/L1个体**
| 动物 | 危险度 | 交互 | 区域种群容量 |
|------|--------|------|------------|
| 兔 | ★☆☆ | 狩猎（需陷阱/弓箭） | 浅山20-50只 |
| 鹿 | ★★☆ | 狩猎，肉值钱 | 浅山10-30只 |
| 野猪 | ★★★ | 危险！需武器 | 深山5-15只 |
| 狼 | ★★★★ | 极危险，夜间出没 | 深山3-8只 |
| 虎 | ★★★★★ | 极罕见，几乎必死 | 深山0-2只 |
| 鱼 | ☆☆☆ | 钓鱼，看技能 | 河流按河段统计 |
| 虾/蟹 | ☆☆☆ | 捞捕 | 河滩按区域统计 |
| 鸟 | ★☆☆ | 用弹弓/网捕捉 | 全区域30-100只 |

**野生动物种群动态（L2统计规则）**
```
每日更新：
  当前数量 = 上日数量 × (1 + 繁殖率 - 狩猎压力 - 自然死亡)

  繁殖率：兔0.05, 鹿0.03, 野猪0.02, 狼0.01, 虎0.005
  自然死亡：0.01（所有种类）
  狩猎压力：取决于区域内猎人数 × 狩猎技能

  数量上限 = 区域容量上限（防止无限增长）
  数量 < 最低阈值 → 触发"此区域动物稀少"提示
```

### 植物

**农作物（可种植）— 玩家田地为L0个体，其余为L2区域统计**
| 作物 | 季节 | 用途 | 亩产参考 |
|------|------|------|---------|
| 稻 | 春种秋收 | 主粮 | 2-4石/亩 |
| 麦 | 秋种夏收 | 主粮 | 1-3石/亩 |
| 蔬菜 | 四季不同 | 食用/售卖 | 按品种 |
| 棉花 | 春种秋收 | 纺织 | 30-50斤/亩 |
| 麻 | 春夏 | 织布 | 50-80斤/亩 |
| 茶 | 常绿 | 采摘制茶 | 20-40斤/亩 |
| 桑 | 常绿 | 养蚕 | 叶量按树计 |

**野生植物（可采集）— 近处L0可交互个体，远处L2统计**
| 类型 | 用途 | 地点 | 重生时间 |
|------|------|------|---------|
| 草药 | 医疗/卖给药铺 | 山林 | 7-15天 |
| 蘑菇 | 食用（有的有毒） | 山林潮湿处 | 3-7天 |
| 野果 | 食用 | 山林 | 5-10天 |
| 柴火 | 烧火做饭 | 山林 | 3-5天 |
| 野花 | 观赏/送礼 | 山野 | 7-14天 |

**区域植被统计（L2）**
```
每区域维护：
  可采集资源密度 = {草药: 数量, 蘑菇: 数量, 野果: 数量, 柴火: 数量}

  每日更新：
    自然再生 = 密度 × 再生率 × 季节修正
    采集消耗 = 采集者数量 × 采集效率

  如果密度接近0 → 暂时无法采集（资源枯竭，需等待再生）
```

## 七、矿物系统（新增）

### 矿物分布

| 矿物 | 分布区域 | 获取方式 | 用途 | 稀有度 |
|------|---------|---------|------|--------|
| 铁矿 | 山脉浅层 | 需镐/锤 | 铁匠锻造 | 常见 |
| 铜矿 | 山脉中层 | 需镐/锤 | 铜器铸造 | 中等 |
| 煤矿 | 山脉浅层 | 需镐 | 冶炼燃料 | 常见 |
| 石料 | 山脚/河边 | 需凿子 | 建筑 | 常见 |
| 黏土 | 河滩 | 徒手可采 | 陶器制作 | 常见 |
| 沙金 | 河流 | 需淘金盘 | 直接换钱 | 稀有 |
| 玉石 | 深山 | 需工具+运气 | 高价售卖 | 极稀有 |

### 矿物生成规则（L2 → L0切换）

```
矿点是固定位置实体：
  - 每个矿点有：类型、剩余储量、再生速率
  - 玩家远离时为L2（仅追踪剩余储量数值）
  - 玩家靠近时具象化为L0（可交互的具体矿点）

  采掘消耗：时间 + 体力 + 工具耐久
  产出 = 基础产出 × 技能修正 × 工具修正 × 随机
```

## 六、社交关系系统

### 关系类型
| 关系 | 来源 | 效果 |
|------|------|------|
| 路人 | 无 | 基础交互 |
| 熟人 | 多次交易/见面 | 可打招呼、小忙 |
| 朋友 | 主动交往、帮助 | 可借小钱、提供消息 |
| 好友 | 长期交往、共患难 | 可借大钱、关键时刻帮忙 |
| 师徒 | 拜师学艺 | 传授技能、师门资源 |
| 婚姻 | 结婚 | 共同生活、财产共享、可生子 |
| 仇人 | 得罪、伤害 | 可能报复、拒绝往来 |

### 交互方式
- **交谈**：日常聊天，增进关系
- **送礼**：快速提升好感，需投其所好
- **帮忙**：NPC有困难时帮助
- **请客**：请吃饭/喝茶
- **拜师**：想学技能需拜师
- **提亲**：关系够了可提亲结婚

### NPC好感度
```
仇视 ←─────── 敌意 ───────→ 冷淡 → 熟络 → 友善 → 亲密
-100           -50           0      30     60     100
```

不同好感度解锁不同交互选项和信息。
