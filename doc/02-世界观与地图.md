# 02 — 世界观与地图

## 时代设定
北宋时期，以汴京（开封）为原型。商业繁荣，夜市兴盛，市民文化丰富。

### 社会背景要点
- **重文轻武**：科举是平民上升主要通道
- **商业发达**：打破坊市制度，商铺临街而设，夜市至三更
- **户籍制度**：主户（有产）vs 客户（无产），影响纳税和权利
- **行会组织**：各行各业有"行"，商人需入行才能正经经营

### 城内区域详解

#### 1. 中心大街
- 游戏主场景，玩家日常经过之地
- 沿街商铺：粮铺、布庄、药铺、铁匠铺、书店、杂货铺
- 路边摊贩：卖炊饼的、卖汤的、算命的、杂耍的
- 流动NPC：乞丐、小偷、巡逻的衙役、赶路的客商

#### 2. 州桥夜市
- 傍晚后开放，热闹至深夜
- 美食摊位：各种宋朝特色小吃
- 可摆摊（如果玩家有手艺/货物）
- 社交场所，能听到各种消息和八卦

#### 3. 瓦舍勾栏
- 娱乐区：说书、杂剧、相扑、傀儡戏
- 可观看表演（花钱，提升心情，获得见识）
- 可拜师学艺（说书/戏曲/武艺）

#### 4. 官衙
- 衙门大堂：报案、诉讼、看告示
- 贡院：科举考试场所
- 衙役房：可应募当差
- 大牢：犯事被关押之地

#### 5. 寺庙区
- 相国寺：汴京最大寺庙，每月有庙会
- 可烧香祈福（心理安慰/触发特殊事件）
- 可与僧人交谈（获得知识/任务）
- 庙会期间商铺云集，热闹非凡

#### 6. 西市 / 东市
- 批发市场，大宗交易
- 客商云集，消息灵通
- 可进货、谈生意

#### 7. 码头区
- 汴河码头，货物吞吐
- 可找搬运工作
- 可搭船出行（未来扩展到其他城镇）
- 鱼市新鲜鱼获

#### 8. 民宅区（北/南）
- 玩家可租住或购买房屋
- 邻居NPC各有故事
- 可被偷窃的高风险区

#### 9. 作坊区
- 各类手工作坊：木工、铁匠、织布、造纸
- 可当学徒学手艺
- 可自己开设作坊

### 第二层：城郊

#### 10. 城外农田
- 水田种稻，旱地种麦
- 可租田/买田务农
- 四季农事不同
- 有农人NPC日常劳作

#### 11. 山林
- 可采集：草药、蘑菇、柴火、野果
- 可狩猎：兔、鹿、野鸡（需要工具和技能）
- 有危险：野兽（狼、虎）、山贼
- 季节变化明显

#### 12. 河流（汴河上游）
- 可钓鱼
- 可淘金（极低概率）
- 夏天可游泳洗澡
- 有水匪风险

#### 13. 邻近村落
- 比城里淳朴，物价比城里低
- 有独特NPC（村长、猎户、药师）
- 可在此租房/买房（便宜但离城远）

## 天气系统

| 季节 | 常见天气 | 影响 |
|------|---------|------|
| 春 | 晴、细雨、雾 | 宜播种；雾天赶路可能迷路 |
| 夏 | 晴、暴雨、酷暑 | 谨防中暑；暴雨影响出行和农田 |
| 秋 | 晴、阴、秋雨 | 收获季节；秋雨连绵影响晾晒 |
| 冬 | 晴、雪、寒风 | 需保暖衣物；大雪封山；物价上涨 |

## 昼夜与时间流逝

| 时辰 | 现代对应 | 游戏内氛围 |
|------|---------|-----------|
| 卯时 | 5:00-7:00 | 天亮，城门开，早市开始 |
| 辰时 | 7:00-9:00 | 众人都忙碌起来 |
| 巳时 | 9:00-11:00 | 商铺全开，适合办事 |
| 午时 | 11:00-13:00 | 午饭时间，部分商铺休息 |
| 未时 | 13:00-15:00 | 午后，茶楼热闹 |
| 申时 | 15:00-17:00 | 傍晚将至，摊贩准备夜市 |
| 酉时 | 17:00-19:00 | 天色渐暗，夜市开张 |
| 戌时 | 19:00-21:00 | 夜生活，瓦舍最热闹 |
| 亥时 | 21:00-23:00 | 夜深，街上人少，危险增加 |
| 子时-寅时 | 23:00-5:00 | 深夜，除夜市外大多歇业，盗贼活跃 |

> 注：每个玩家动作消耗一定时辰（如吃饭=半个时辰，干活=两个时辰），不操作则时间静止。
