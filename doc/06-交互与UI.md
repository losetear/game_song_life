## 核心交互模式

### 1. 场景描述 → 选项选择

**原则**：每个场景都有文字描述 + 3-6个按钮选项

```
场景示例 — 布庄内

─────────────────────────────────────
你推开布庄的门，里面挂满了各色绸缎布匹。
王掌柜正在柜台后面拨算盘，抬头看到你，
露出一丝笑容。

"哟，稀客啊！今天想看点什么？"

架上有：粗麻布、细棉布、青绸、苏绣残片。
角落里还堆着几匹打折的旧布。
─────────────────────────────────────

[买粗麻布 50文]  [买细棉布 200文]  [买青绸 500文]
[问苏绣残片]     [看打折旧布]      [和王掌柜闲聊]
[讨价还价]       [离开布庄]
```

### 2. 对话系统

```
对话示例 — 与刘寡妇聊天

─────────────────────────────────────
刘寡妇边擦桌子边说：

"哎你知道吗，昨晚张三又被钱捕头抓了！
听说是偷了东边客栈一个客商的包袱。
那客商气得不行，非要告到县太爷那儿去。"

她压低声音："不过我听说，那张三好像是
受人指使的……具体是谁，我可不敢乱说。"
─────────────────────────────────────

[追问是谁指使的]  [打听张三平时和谁来往]
[转移话题聊别的]  [道谢离开]
```

### 3. 工作交互

```
工作示例 — 农田种稻

─────────────────────────────────────
你站在自家薄田边上，春日的阳光正好。
田里的水已经放好，泥土松软。

根据季节（春季），现在可以：
- 翻地整田（消耗体力，提升产量）
- 撒秧播种（需要稻种）
- 引水灌溉
- 休息等待

你的体力还剩 60%，今天还能干不少活。
─────────────────────────────────────

[翻地整田 - 消耗1时辰 - 体力-20%]  [撒秧播种 - 需要稻种]
[引水灌溉 - 消耗半个时辰]          [在田埂上歇会儿]

```

## 响应式设计

### 移动端适配
- 左侧状态栏收起为顶部折叠条
- 迷你地图隐藏，通过按钮展开
- 选项按钮改为纵向排列
- 底部功能栏改为底部抽屉

### 交互反馈
- 每次选择后，文本区平滑过渡到新内容
- 状态变化（如金钱减少）在状态栏有短暂高亮
- 重要事件（被偷、生病等）有醒目提示
- NPC对话带角色标识（不同颜色或头像）
