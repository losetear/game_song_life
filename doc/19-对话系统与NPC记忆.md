# 一、对话系统

## 1.1 架构

对话不是预写脚本，而是**模板 + 动态变量 + 条件分支**。

| 组成 | 说明 |
|------|------|
| 开场白 | 根据好感度/天气/时间等条件匹配不同文案 |
| 分支选项 | 对话按钮，每个有显示条件 |
| NPC回复 | 每个选项对应回复文案 |
| 效果 | 选择后的好感变化/物品变化/事件触发 |
| 嵌套分支 | 对话可多层嵌套 |
| 退出标记 | 选择后结束对话 |

## 1.2 条件匹配

| 条件类型 | 说明 | 示例 |
|----------|------|------|
| 好感度 | 根据NPC对玩家好感 | 好感>70→"哎呀，老朋友来了！" |
| 天气 | 根据当前天气 | 下雨→"快进来避避雨" |
| 时间 | 根据当前时辰 | 晚上→"这么晚还没歇？" |
| NPC状态 | 根据NPC当前行为 | 忙碌→"等一下，我先把手头的活干完" |
| 玩家技能 | 根据玩家是否学了某技能 | 会点茶→"来泡一壶？" |
| 近期事件 | 根据最近发生的事件 | 刚下了暴雨→"这雨可真大" |

## 1.3 动态变量

| 变量 | 说明 |
|------|------|
| {player.name} | 玩家角色名 |
| {npc.name} | NPC名 |
| {weather.desc} | 天气描述 |
| {time.desc} | 时间描述 |
| {item.price} | 商品价格 |
| {shop.newArrivals} | 店铺新到商品 |

## 1.4 对话示例——李掌柜（绸缎庄）

**开场白（按好感分级）**：

| 好感范围 | 开场白 |
|----------|--------|
| 70+ | "哎呀，{player.name}！来了来了，快进来坐。正好新到了一批上好的苏绣，你一定会喜欢。" |
| 40-69 | "{player.name}啊，今天来有什么事？看看布料？" |
| 10-39 | "客官，随便看看，有什么需要说一声。" |

**对话分支**：

| 选项 | 条件 | NPC回复 | 效果 |
|------|------|---------|------|
| 看看布料 | 无 | "这边请，今天有{shop.newArrivals}，品质不错。" | 进入商品界面 |
| 聊聊天 | 好感>40 | 根据最近事件随机聊天 | 好感+2 |
| 你这里招人吗？ | NPC需要伙计 | "正好缺个伙计……" | 进入应聘流程 |
| 告辞 | 无 | "慢走。" | 结束对话 |

**砍价子分支**：

| 选项 | 条件 | NPC回复 |
|------|------|---------|
| 这匹丝绸怎么卖？ | 无 | 按好感分级报价（好感>60打85折） |
| 能便宜点吗？ | 砍价技能>20 | 砍价判定，成功降价5-15% |
| 买了 | 有钱 | "好嘞，包好给你。" |
| 再看看 | 无 | "不急，慢慢看。" |

## 1.5 对话内容规模

| 类别 | 预估数量 |
|------|----------|
| 通用开场白模板 | 50条 |
| 店铺对话模板 | 100条 |
| 闲聊话题模板 | 200条 |
| 任务相关对话 | 100条 |
| 节日对话模板 | 50条 |
| **总计** | **~500条** |

# 二、NPC 记忆系统

## 2.1 记忆结构

| 字段 | 说明 |
|------|------|
| 事件ID | 关联的事件 |
| 摘要 | 记忆摘要（自然语言） |
| 重要度 | 1-10 |
| 情感标签 | 正面/负面/中性 |
| 关于谁 | 目标实体 |
| 发生时间 | 游戏时间 |
| 可传播 | 是否可以告诉别人 |

## 2.2 记忆类型与保留

| 类型 | 保留时间 | 上限 | 示例 |
|------|----------|------|------|
| 短期 | 1-2游戏天 | 50条 | "今天有人在街上掉了钱袋" |
| 长期 | 永久 | 100条 | "张三三年前救过我的命" |
| 态度 | 永久更新 | 每实体1条 | "对玩家好感72：好友" |
| 可传播 | 直到传遍 | 不限 | "城东王家茶坊出了新鲜事" |

## 2.3 好感度分级

| 分值 | 标签 | 可解锁交互 |
|------|------|-----------|
| 0-20 | 仇人 | 只有道歉/逃跑 |
| 21-35 | 厌恶 | 基本交互 |
| 36-50 | 陌生人/认识 | 基本交互 |
| 51-65 | 熟人 | 闲聊/打听消息 |
| 66-80 | 朋友 | 借钱/学手艺/帮忙 |
| 81-95 | 挚友 | 几乎全部 |
| 96-100 | 恩人 | 全部+特殊 |

## 2.4 好感变化规则

| 行为 | 好感变化 |
|------|----------|
| 帮忙/送礼/友好对话 | +3~+10 |
| 拒绝/无视/小冒犯 | -3~-10 |
| 救命/重大帮助 | +20~+30 |
| 伤害/偷盗/重大冒犯 | -20~-30 |
| 长时间不互动 | 每月±1（回归中性） |

## 2.5 口碑传播

| 规则 | 说明 |
|------|------|
| 好事传播 | 好感>70的NPC会向好友提及，好友好感+2 |
| 坏事传播 | 好感<30的NPC向更多人传播，范围更大 |
| 传播衰减 | 每次传播效果减半 |
| 地域限制 | 同一城市内，不跨城 |
| 时间衰减 | 旧消息逐渐被遗忘 |

# 三、NPC 生命周期

## 3.1 生成

| 区域需求 | 生成规则 |
|----------|----------|
| 每个店铺 | 1个掌柜 + 1-3个伙计（按店铺规模） |
| 每个居民区 | 按容量生成居民 |
| 每个功能区 | 按功能生成对应职业NPC |

**姓名生成**：使用宋代常见姓名库

| 性别 | 姓氏库 | 名字库 |
|------|--------|--------|
| 男 | 王/李/张/赵/陈/刘/周/吴/郑/孙/钱/林/黄 | 福来/大有/国柱/承恩/文昌/文远/明德/仁杰/世安/建平 |
| 女 | 同上 | 翠娘/秀兰/桂花/素芬/巧云/玉兰/月娥/淑贞/银屏/春娘 |

## 3.2 死亡处理

| 步骤 | 说明 |
|------|------|
| 1. 通知关系人 | 按关系亲密度降低心情（家人-30/朋友-10） |
| 2. 生成记忆 | 所有关系人在长期记忆中记录 |
| 3. 财产继承 | 子女→配偶→兄弟姐妹→充公 |
| 4. 店铺继承 | 继承人获得店铺所有权 |
| 5. 关系继承 | 继承人继承部分社会关系和恩怨 |
| 6. 纪念事件 | 触发"npc_died"事件 |
| 7. 延迟移除 | 等待系统处理完毕后移除实体 |

## 3.3 日程调度

### 工作日

| 时段 | 活动 | 受影响因素 |
|------|------|-----------|
| 卯时(5-7) | 起床 | 年龄（老人早起） |
| 辰时(7-9) | 通勤去工作 | 距离 |
| 辰时-午时 | 工作 | — |
| 午时 | 午饭 | 性格（节俭→带饭/大方→下馆子） |
| 未时-申时 | 继续工作 | — |
| 酉时 | 收工 | — |
| 酉时-戌时 | 自由时间 | 性格（好社交→茶坊/好学→读书/其他→在家） |
| 亥时 | 就寝 | — |

### 假日

| 时段 | 活动 |
|------|------|
| 巳时 | 晚起 |
| 午时-申时 | 节日活动/逛街 |
| 戌时-亥时 | 夜市/社交 |

**日程可被事件打断**：下雨→收摊 / 危险→逃跑 / 玩家交互→响应
